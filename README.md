WeatherCompose

Android приложение погоды, построенное на Jetpack Compose и Retrofit. Отображает текущую погоду, почасовой прогноз и прогноз на 5 дней.
Возможности

    Текущая погода с температурой, условиями и городом/временем

    Вкладка почасового прогноза с температурой и условиями

    Вкладка ежедневного прогноза с максимальной/минимальной температурой на 3 дня

    Диалог поиска города для смены местоположения

    Функция обновления данных

Технологии

    Jetpack Compose

    Retrofit & Gson для API-запросов

    ViewModel с StateFlow

    Интеграция с WeatherAPI.com

Использует WeatherAPI.com для получения данных о погоде. Требуется API-ключ (доступен бесплатный тариф).

Архитектура

    Паттерн MVVM

    Слой Repository для абстракции данных

    Функции расширения для преобразования моделей

    Внедрение зависимостей с ViewModelFactory
